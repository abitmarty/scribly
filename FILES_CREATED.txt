DESIGN DECISIONS:
=================

✅ Blade Templates - Easy to customize, no frontend build step
✅ Tailwind CDN - No npm install, fast for landing pages
✅ Alpine.js CDN - Lightweight interactivity, no build tools
✅ File Storage in storage/ - Secure, not publicly accessible
✅ Synchronous Processing - Shared hosting friendly
✅ MySQL Database - Easy backup, Cloud86 standard
✅ SMTP Email - Works with Cloud86 cPanel

QUICK START:
============

Local Testing:
  php artisan serve
  Visit: http://localhost:8000

Deploy to Cloud86:
  1. Create MySQL database in cPanel
  2. Upload files to /home/username/scribly/
  3. Set document root to /home/username/scribly/public
  4. Create .env file with database credentials
  5. Run: php artisan migrate --force
  
  See SETUP.md for detailed instructions.
